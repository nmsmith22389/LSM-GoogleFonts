### NOTE: This addon only works with addons that support `Lib-SharedMedia-3.0`

## Description
**LSM-GoogleFonts** is a font pack for WoW that gives you some popular google fonts.

**Fonts Included:**

* Anton
* Audiowide
* Exo2
* Geo
* Monda
* Nova Square
* Open Sans Condensed Bold
* Orbitron
* Quantico
* Roboto
* Teko
* Titillium Web
* Ubuntu
* VT323
